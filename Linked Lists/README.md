# Linked Lists

Applying a Doubly-Linked List to manipulate PNGs.

Input and Output in /images.

Run:
- ./main
- ./testBlock
- ./testChain

Example:
- Chain
  - 60 columns
  - 70 rows
  - reverse()
  - rotate(3)

- Block
  - flipVert()
